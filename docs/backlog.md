# Backlog

Идеи и пожелания, которые не входят в MVP. Будут рассмотрены после первых пользователей.

---

## [x] B-1. Год рождения игрока и возрастное ограничение раунда

**Идея:** фиксировать год рождения игрока (`birth_year`) и добавить опциональное возрастное ограничение в параметры раунда.

**Детали:**
- `players`: добавить поле `birth_year` (integer, nullable)
- `rounds`: добавить поле `age_restriction` (JSON или отдельные поля `age_min` / `age_max`, nullable)
- При добавлении команды в раунд — опциональная валидация: возраст всех игроков команды попадает в ограничение
- Возраст считается на дату начала турнира (`tournaments.date_start`)

**Применение:** возрастные категории (юниоры, ветераны), разделение по группам U17 / U19 / 35+ и т.д.

---

## [x] B-2. Фильтрация, поиск и сортировка игроков и команд

**Идея:** добавить возможность быстрого поиска и упорядочивания списков на экранах `/players` и `/teams`.

**Детали:**
- `/players`: текстовый поиск по фамилии / имени / клубу, сортировка по фамилии (A→Z / Z→A) и по клубу
- `/teams`: фильтр по категории (MS / WS / MD / WD / XD), текстовый поиск по названию команды или фамилии игрока
- Реализуется на стороне клиента (фильтрация уже загруженного списка) — без изменений в БД и IPC
- Состояние поиска/фильтра не сохраняется между сессиями (сбрасывается при переходе)

---

## [x] B-3. Время начала матчей в расписании

**Идея:** в настройках турнира задавать время начала матчей для каждого игрового дня.

**Детали:**
- Новый экран или секция в настройках турнира: список игровых дней (диапазон `date_start`…`date_end`) с полем «Start time» для каждого дня
- Хранить в отдельной таблице `tournament_day_settings (id, tournament_id, date, start_time, match_duration)` или как JSON-поле в `tournaments`
- Экран расписания (`/tournaments/:id/schedule`) использует `start_time` конкретного дня вместо хардкода `09:00` при вычислении следующего свободного слота (`getNextSlot`)
- Также использовать `match_duration` вместо константы `MATCH_DURATION = 60`

**Применение:** турниры с нестандартным расписанием (ранний старт, несколько игровых дней с разным временем начала).
